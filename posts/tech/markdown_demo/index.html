代码#

  • 要创建代码块，请将代码块的每一行缩进至少四个空格或一个制表符。
`短代码`  单个反引号扩起来
```
	echo "hello world"
	echo "day day up"
```

echo "hello world"
echo "day day up"

分割线#

  • 要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (---) 或下划线 (___) ，并且不能包含其他内容

  • 为了兼容性，请在分隔线的前后均添加空白行

    ***
